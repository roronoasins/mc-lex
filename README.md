# Modelos de Computación UGR
> Escuela Técnica Superior de Ingeniería Informática y Telecomunicaciones, Universidad de Granada

Practicas realizadas en la asignatura Modelos de Computación - Dpto. de Ciencias de la Computación e I.A. - en la Universidad de Granada, curso 2018-2019.

## Herramientas utilizadas

- Lex Scanner Generator
- C++
- [Atom.io](https://atom.io/)

## Descripción

Pensar un problema original de procesamiento de textos. Para la resolución de este problema debe ser apropiado el uso de Lex, o sea, se debe resolver mediante el emparejamiento de cadenas con expresiones regulares y la asociación de acciones a cada emparejamiento.

### Mi solución

Para la resolución de este ejercicio hice un programa el cual extrae la información de [*Instant-Gaming*](https://www.instant-gaming.com/es/) para un videojuego concreto utilizando su código fuente html.

- Título de la entrega
- Portada
- Enlace del tráiler
- Categorías
- Requisitos de sistema
- Descripción
- Precio

> Funciona para la versión en castellano de la página, para otras como francés no funciona debido a la aparición de caracteres no tomados en cuenta.
